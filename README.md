# Explore Sri Lanka 🇱🇰

Welcome to **Explore Sri Lanka**, your ultimate guide to discovering the wonders of the Pearl of the Indian Ocean. This website offers curated travel packages, locally guided tours, and unforgettable experiences across the beautiful landscapes of Sri Lanka.

## 🌐 Live Demo
Check out the live website: [https://exploresrilankaa.netlify.app/](https://exploresrilankaa.netlify.app/)

## ✨ Features
- **Curated Destinations**: Explore top attractions like Sigiriya, Ella, Kandy, Galle, and Yala.
- **Travel Packages**: Choose from specialized tours including Beach Escapes, Hill Country Retreats, and Wildlife Safaris.
- **Responsive Design**: Optimized for a seamless experience across desktop, tablet, and mobile devices.
- **Interactive Gallery**: A visual showcase of Sri Lanka's stunning landscapes and heritage.
- **Booking Integration**: Easy-to-use contact form for tour inquiries and bookings.

## 🚀 Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Fonts**: Inter (Google Fonts)
- **Icons**: Font Awesome 6.5.1
- **Deployment**: Netlify

## 📁 Project Structure
- `index.html`: Main structure and content of the website.
- `style.css`: Custom styling and responsive design.
- `script.js`: Interactive elements and navigation logic.
- `assets/`: Directory containing images, logos, and other media.

## 🛠️ Local Development
To run this project locally:
1. Clone the repository.
2. Open `index.html` in your preferred web browser.
